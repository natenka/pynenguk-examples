============
Installation
============


Stable release
--------------

To install project3_try, run this command in your terminal:

.. code-block:: console

    $ pip install project3_try

This is the preferred method to install project3_try, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for project3_try can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/gituser/project3_try

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/gituser/project3_try/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/gituser/project3_try
.. _tarball: https://github.com/gituser/project3_try/tarball/master
